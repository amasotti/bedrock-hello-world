# AWS Bedrock Hello World

This is a simple example of how to invoke a Bedrock model using the Kotlin SDK.


## Usage

1. Clone the repository

```bash
  git clone  git@github.com:amasotti/bedrock-hello-world.git
```

2. Build the project

```bash
  ./gradlew build
```

3. Run the project

```bash
  ./gradlew run
```

## Tooling

- Kotlin v2.1.10
- Gradle v8.12
